# Work Order — L1: Single Agent + One Tool Call

## Goal
Introduce a minimal "agent" runner and a single tool (`run_tests`) that returns
structured JSON output. Keep it deterministic in `--dry-run` mode.

## Acceptance Criteria
- `python -m agent_lab.l1_agent --dry-run` exits 0 and prints valid JSON
- `python -m agent_lab.l1_agent run-tests` executes `pytest -q` and prints valid JSON
- JSON includes: status, command, exit_code, stdout_tail, stderr_tail
- `pytest -q` passes
- CI passes

## Non-goals
- No multi-agent handoffs yet
- No tracing yet
- No network calls required
